Bienvenido,Este programa corresponde al Certamen N°2 extendio de Manuel Carreño Puga-Andres Lopez-Claudio Rojas.

Para la ejecucion del programa es necesario que los 3 archivos esten dentro de la misma carpeta 
dentro del PC en que se revise el programa.

Luego,por medio de un Terminal o Consola(Cmd) ingresar a dicha carpeta de la forma cd c:\usuario\carpeta
una vez nos aseguramos que nos encontramos dentro de la carpeta con los comandos dir o ls,
Ejecutamos el programa con el siguiente comando: "go run main.go EjemploSeccion2.go banco.go"

Una vez el programa se este ejecutando en el terminal se desplegara un menu en el cual el usuario podra escoger
entre revisar el Certamen2 2.a y 2.b y el Certamen2 Extendido ingresando de forma manual 
por la terminal el numero 1 o 2 respectivamente.

Una vez escogidas una de las opciones se confirma la opcion con el boton ENTER el cual comenzara la ejecucion 
de la parte del Certamen2 seleccionada.

Si escoge el Certamen2 Extendido se recomienda esperar a que la ejecucion finalise para apreciar bien el flujo
de los datos dado su constante movimiento.

El unico punto de la rubrica no desarrollado para el Certamen2 Extendido fue que el usuario ingresara de manera manual
la cantidad de cajeros y el timpo de arribo,debido a que,por la manereja de accionar las corrutinas 
la cola,los cajeros y la cantidad de clientes deben ser multiplos,y el tiempo de arribo de clientes y de atencion 
al cliente fue realizado de manera randomica como se solicito en la primera version del enunciado de este Certamen2.
